# Mobile Money App

This is a mobile money application built with Python and Flask.  
It simulates basic mobile money operations like checking balance, depositing money, and sending money between users.

## Features
- Check user balance
- Deposit money
- Send money to another user
- Simple REST API for interaction

## Requirements
- Python 3.x
- Flask
